# 🎬 Netflix Movie Recommendation System

A **Content-Based Movie Recommendation System** built using Python that suggests similar Netflix movies/shows based on their descriptions.

This project uses **Natural Language Processing (TF-IDF)** and **Cosine Similarity** to recommend the top 5 similar titles along with visual insights.

---

## 🚀 Features

* 🎯 Content-based recommendation system
* 🔍 Uses **TF-IDF Vectorization** for text analysis
* 📊 Calculates similarity using **Cosine Similarity**
* 🎬 Displays **Top 5 recommended movies/shows**
* 📈 Graph visualization of recommendations
* 📊 Trend analysis of Netflix releases over years
* 🧾 Displays movie details (Director, Year, Description, Country)
